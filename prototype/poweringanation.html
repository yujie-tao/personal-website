Powering A Nation

Prototype

The goal of this project to create a well-designed, easy-to-navigate, flawlessly scripted multi-page website for Powering A Nation. I created the wireframes, designed the mockups and coded it in HTML and CSS. There are five pages in this site: Home, Wind, Coal, Electric, Credits. All of them are structured in Bootstrap, so they are built in grids and responsive to different screen sizes.
